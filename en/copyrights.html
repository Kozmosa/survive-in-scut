Copyrights

This page is meant to summarize the licensing status of the project text, attachments, third-party assets, and embedded resources.

There is currently a maintainer-facing inconsistency that should be resolved soon: the repository root LICENSE is CC0 1.0, while the homepage and some documents still mention CC-BY-SA 2.0 or page-level exceptions. Until that is clarified, readers should treat reuse rules conservatively.

Suggested conservative interpretation for now:

  • The root LICENSE is the repository-level default statement.
  • If a page includes its own source, restriction, or exception note, that page-level note should take priority.
  • Third-party images, PDFs, embedded tools, and external resources should not be assumed to share the same license as the handbook text.
  • If reuse, redistribution, or commercial use matters, verify the original source instead of relying on a single sentence in this repository.
