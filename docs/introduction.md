# 简介

这不是学校官网，也不是一份保证永远正确的“标准答案”。

这是一个由学生自发维护的华南理工大学生存手册。它更像一套经验索引：把新生、在校生和已经踩过坑的人手里的信息整理出来，让后来的人少走一点弯路。

## 这本手册是什么

它主要解决三类问题：

- **到校与适应**：第一次来华工，先做什么、去哪看、哪些入口最重要。
- **学习与规划**：课程、竞赛、SRP、转专业、升学与科研入门从哪里开始。
- **生活与求助**：吃饭、出行、就医、周边信息、常见坑和经验判断。

它不追求百科全书式覆盖，而是优先收录“新同学真的会用到”的信息。

## 这本手册不是什么

- 不是学校官方规则汇编
- 不是对所有问题都负责的行动指南
- 不是替你做决定的保姆式教程
- 不是任何年份都自动不过期的资料库

尤其是时间敏感内容，例如迎新安排、校巴、办事窗口、地图细节、课程政策、转专业口径，默认都应该再核对一遍官方通知。

## 适合谁看

最直接的目标读者是：

- 刚录取、准备入校的本科新生
- 想快速建立校园生活地图的在校生
- 正在准备竞赛、SRP、转专业、科研或升学的同学
- 想给后来者补充经验的学长学姐

## 建议怎么用

如果你是第一次打开，建议不要从头顺着读，而是按你的当前问题走：

- 刚拿到录取通知：先看 [启程](/get-started)
- 想认路：看 [设施](/infra/)
- 想解决吃饭和日常：看 [生活入口](/life/time/hemc_gzic)
- 想知道课程和学习资源：看 [学习入口](/learn/curricular/common_basic_lessons)
- 想看升学、保研、考研、出国：看 [飞跃入口](/beyond/mainland/recommend_graduate)
- 想了解就医、安全与底线问题：看 [立命入口](/health/alive_first)

把它当作“入口导航 + 经验索引”，而不是逐字背诵的手册，会更有用。

## 哪些内容最容易过期

请默认下面几类内容需要二次确认：

- 迎新时间、接站安排、校门开放和车辆规则
- 学校系统入口、表单、二维码、群号
- 课程安排、培养方案、学分口径、政策细则
- 校巴、饭堂、商铺、周边营业情况
- 报销、医保、校医院、办事窗口和办公时间

如果某条信息对你很重要，优先去找“最近一次更新时间”或“适用学年”。如果没有，就把它当作经验参考，而不是最终依据。

## 如果你发现错误，怎么贡献

最简单的贡献方式有两种：

- 直接提交 PR
- 把错误、过期内容、补充材料发给维护者或编辑部

更完整的说明请看：[参与贡献](/others/contributing)

如果你准备提 PR，建议先看 [TODO 汇总](/others/todo)，优先从公开可见、明显过期或长期空白的页面开始补。

## 维护上的基本共识

这本手册的长期价值，不在于“每页都写得很满”，而在于：

- 高流量入口页持续可用
- 过期内容被及时修掉
- 贡献者容易补充内容
- 读者能分清“经验”与“官方规则”

只要这四件事做得住，它就会一直有用。
