# 维护路线图

本手册优先维护会直接影响新生决策和日常生活的信息，并持续补齐缺失主题。

## 当前优先级

1. 每学年核对迎新、校巴、地图、就医、转专业和周边信息等高时效页面。
2. 补充中英文页面的内容缺口，并标注暂未验证的信息。
3. 修复失效链接、补齐图片与附件的来源说明。
4. 将可复用的经验沉淀为清晰、可维护的 Markdown 文档。

## 如何参与

请先浏览 [TODO 汇总](/others/todo)，选择一项可完成的工作；随后参照[贡献指南](/others/contributing)提交 PR 或反馈。

如发现内容过期、链接失效或信息不准确，也欢迎直接提出 Issue。
