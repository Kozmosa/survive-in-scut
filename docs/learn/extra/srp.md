# SRP

SRP，全称为 Student Research Plan，是一项华南理工大学所特有的校级本科生科研项目。参与并成功完成结题可获得2分（半年期 SRP） / 4分（一年期 SRP）创新分。

大学生创新创业训练平台（需要校园网打开）：

- [华南理工大学 大学生创新创业训练计划管理系统](https://211.66.34.2/xm/CXCY/SCUT)

## 项目要求

每位教师每年指导SRP不超过1项，每个项目参与学生为2-7人，遴选对象应优先为一、二年级本科生，**原则上每位学生在校期间只能参加1项SRP**。

## 项目时间

每年4月——次年3月

## 学生报名

报名时间：2026年3月18日00:00——2026年3月22日24:00

线上申报，平台为“华南理工大学大学生创新创业训练计划管理系统”

网址：

https://211.66.34.2/xm/CXCY/SCUT （需要校园网打开）

https://211-66-34-2.webvpn.scut.edu.cn/xm/CXCY/SCUT （webvpn）

- 登录系统，在“个人中心”——“通用”完善邮箱和手机信息。

- 点击“选题管理”——“学生选择选题”，浏览全校发布的选题，根据个人能力和兴趣选择选题，填写“选题说明”，视需要上传附件，点击“提交”。

- 学生可在“选题管理”——“学生选择选题”，查看已报名的选题信息及入选状态。

## 提示

tips：

- 每位学生最多同时在选5个选题，但最终只能参加1个选题。若报名了多个选题，你将进入最先选择你的教师的项目。

- 建议报名选题后主动联系老师，因为：
  - 1、可以自我介绍，询问项目相关要求和工作细节；
  - 2、较热门的项目大概率会需要面试；
  - 3、部分老师不看系统报名，只选通过邮箱或微信联系他的学生。

- SRP 的工作强度随项目而异：大多数项目不会对参加项目的本科生提出过于苛刻的要求，按时按量完成任务、提交结题报告，即可顺利结题完成项目、得到创新分；少数项目（以本研究内容为基础，后续继续申报大创、挑战杯等比赛；或以发表科研论文为目标、以导师真实科研项目中的课题立项的项目）提出的要求、以及后续的工作强度会比较高。
- 建议仔细阅读平台上的项目简介，并发邮件与导师沟通项目的要求细节确定自己是否适合目标项目。


## Q&A

**Q1**：为什么我报名之后那个项目不见了？
**A**：在选题界面将**选择状态**从”未选择“改成”已选择“。

**Q2**：附件一定要提交吗？要提交什么？
**A**：不一定。如有能够展现你优势的文件或证明，可作为附件提交。

**Q3**：怎么判断这个项目“*水不水*”？
**A**：通过两个方面了解：
1、向学长学姐寻求推荐；
2、阅读该项目的**学生要求**：没有对**某项能力**和**空余时间**提出硬性要求的，强度可能较低。请自行判断。

**Q4**：项目应该报本学院的还是非本学院的？
**A**：没有要求，自行抉择。本学院的你可能更熟悉，也有些老师会更鼓励跨学院报名的学生。

**Q5**：在校期间能报多次吗？
**A**：**原则上** 每位学生在校期间只能参加1项 SRP。一般前一年的 SRP 项目结题之前，次年的 SRP 就开始报名，所以很难两年连着报。可以尝试大一一次，大三一次。

**Q6**：同时被多个老师接受了会怎么样？
**A**：不会出现这种情况。在平台上先被哪个老师接受就进入哪一个组。

**Q7**：SRP 和大创的关系？
**A**：部分 SRP 后续会继续申报大创项目，或前身是大创项目，进此类 SRP 可以顺便参加大创。

**Q8**：SRP 项目会有校区不同的问题吗？
**A**：一般线上开组会，要求中没写线下组会 / 线下实验室值班的项目问题不大。

**Q9**：一个 SRP 项目招多少人？
**A**：2-7人。

**Q10**：SRP 项目和进实验室冲突吗？
**A**：不冲突。
