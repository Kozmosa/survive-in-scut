# 启程，华南理工大学

> 本页只保留跨年份相对稳定的新生入校流程与判断方法。
> 报到日期、接站安排、校门开放、缴费口径等每年都会变，请以当年录取通知书、迎新系统、学院通知和学校官方公告为准。

如果你是第一次来华工，这一页的目标不是替你做全部决定，而是帮你少踩几个最常见的坑：先做什么、哪些东西不能忘、哪些信息最容易过期、出了问题该去哪里确认。

## 到校前，先把四件事做完

### 1. 核对官方入口

优先确认这些入口当前是否可用：

- 迎新系统：<https://freshman.scut.edu.cn/>
- Mypass / 到校报到相关入口：<https://enroll.scut.edu.cn/door/index_h5.html>
- 校园地图：<https://map.scut.edu.cn/login_home.html>

如果链接、流程或字段和本页描述不一致，以当年官方入口为准。

### 2. 准备证件与必要材料

建议至少准备：

- 录取通知书
- 身份证原件
- 若身份证办理有特殊情况，则准备户口簿等替代证件
- 手机号、银行卡、常用支付方式
- 个人近期证件照电子版与纸质版若干
- 学院要求的其他专项材料

不要把所有原件都塞进托运行李。证件、手机、充电器、少量现金、应急药物，最好全程随身。

### 3. 先搞定账号与数字系统

在出发前尽量完成这些事情：

- 激活统一认证账号
- 登录迎新系统，检查个人信息采集是否完成
- 看懂报到当天哪些环节必须在线完成
- 提前加入学院、班级或辅导员发布的官方通知渠道

真实体验里，很多人不是卡在路上，而是卡在“账号没激活”“通知没看到”“系统入口找不到”。

### 4. 做一个“第一天最小清单”

建议把下面几件事记在手机备忘录里：

- 找到宿舍并完成入住
- 确认报到注册是否成功
- 领卡、领钥匙、领物资时检查有没有漏项
- 确认学院后续通知群和新生群是否已经进全
- 晚上把第二天要用到的证件和路线再检查一遍

## 到校当天，先后顺序怎么排

一个实用顺序通常是：

1. 先到宿舍或迎新点，确认你的人已经“落地”。
2. 再处理线上报到、身份核验、宿舍钥匙、一卡通等硬性事项。
3. 然后再考虑买生活用品、逛校园、认识同学、参加社交活动。

别一上来就把时间花在办卡、社团宣传、推销摊位或无关紧要的排队上。第一天最重要的是把“必须完成”的环节全部收口。

## 来校交通怎么判断

这部分信息每年会微调，所以这里不直接写死某一年的接站和校门安排，只给判断方法。

### 五山校区

适合先确认：

- 离你最近的是五山站、天河客运站，还是公交站
- 你到的是南门、东门、北门还是北区入口
- 宿舍在本部还是北区

### 大学城校区

适合先确认：

- 你到的是大学城南站还是周边公交站
- 你是先去生活区宿舍，还是先到教学区迎新点
- 行李车和家长车辆是否有临时停放限制

### 广州国际校区

适合先确认：

- 你是从地铁换乘公交，还是网约车直达
- 南门、北门、西南门、西北门里，哪一个是当前通知要求使用的入口
- 你到达时间是否处于学校统一接驳或服务时段内

## 入住和报到时最容易踩的坑

### 不要默认“到了学校就算报到成功”

很多环节是分开的：人到了、宿舍住进去了、系统登记了、学院确认了，这几件事不一定同时完成。当天一定要亲眼确认系统状态或学院口径。

### 不要把“学长学姐说的”当成最终规则

经验可以参考，但缴费、缓缴、绿色通道、党团关系、户口迁移、档案邮寄这类事项，最终都要看当年正式通知。

### 不要在第一天做太重的采购

先住下，看看宿舍尺寸、收纳空间、室友分工，再决定哪些东西要买。很多新生第一天买太多，最后搬运和退货都很痛苦。

### 不要轻信推销和代办

凡是“帮你办卡”“内部渠道”“只限今天”“先交钱再说”的，默认提高警惕。遇到不确定的情况，优先问辅导员、班助、学院迎新点或学校官方渠道。

## 如果你家庭经济压力比较大

重点先看当年的“绿色通道”、助学贷款、缓缴学费、临时困难补助相关通知。原则上，这类事项越早确认越好，不要拖到错过办理窗口。
