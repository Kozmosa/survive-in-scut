# 版权声明

本页用于汇总本项目内容、附件、第三方素材和嵌入资源的授权信息。

当前仓库存在一个需要维护者尽快澄清的现实问题：仓库根目录 `LICENSE` 是 `CC0 1.0`，但首页和部分文案又写成了 `CC-BY-SA 2.0` 或“另有注明”的口径。对于读者和贡献者来说，这会直接影响转载、二次创作和素材复用的边界判断。

在正式统一口径前，建议按下面的保守原则理解：

- 仓库根目录 `LICENSE` 代表仓库级默认授权声明。
- 页面内单独写明来源、限制或例外的内容，应优先遵守页面自身说明。
- 引用的第三方图片、PDF、嵌入页面、交互工具和外链资源，不应自动视为与仓库正文处于同一授权范围。
- 对转载、商用、二次分发有疑问时，应先核对原始来源，而不是只看本仓库的一处文案。
