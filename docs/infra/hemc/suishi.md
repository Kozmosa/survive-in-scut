# 穗石村

穗石村紧邻大学城校区，从图书馆与 A1 教学楼之间的道路直行约 3 分钟出校门即达，是日常外食和采购的好去处。

## 餐饮

- **面食/饭类**：震亚牛肉面、隆江猪脚饭
- **正餐/聚餐**：北调儿铜炉鸡锅、东北菜、武汉小炒、仙庙烧鸡、鸡公煲
- **饮品**：蜜雪冰城、茶百道、幸运咖

## 日用采买

穗石村有比校内西亚兴安综合超市规模稍大的私营超市，但没有大润发、沃尔玛之类的大型连锁超市。日常水果、零食和日用品均可在此采购。

## 交通接驳

从穗石村返回学校由**东门**进入即可。前往地铁站需穿过整个教学区，相比从生活区西一门前往大学城南站，穗石村到地铁站并不算方便。

## 租房

穗石村有可供学生租住的房源，适合考研备考、实习等情况。详细租房经验欢迎有相关经历的同学补充。
