# 五山周边

五山校区地处广州市区，校门外便是繁华的天河区，周边餐饮、购物、娱乐选择丰富。

## 餐饮

- **五山地铁站周边**：从东门或东侧门出校即达五山站附近，周边遍布各类餐馆和小吃店
- **天河客运站周边**：从北区博学楼旁的北区北门出，步行数百米即到天河客运站，附近有天河新天地等商圈，餐饮选择丰富

## 租房

五山校区内的**西秀村**有教工宿舍可供出租，但价格相对较高。更多周边租房经验欢迎有相关经历的同学补充。

## 交通接驳

### 地铁

- **五山站（3号线）**：西区、南区、东区最方便的地铁站，从东门或东侧门出即达
- **天河客运站（3号线、6号线）**：北区最方便的地铁站，从北区北门出步行数百米即达

### 跨校区交通

往返大学城校区和国际校区最方便且廉价的方式是乘坐**校巴**。校巴从校内的图书馆等站点出发，经由快速路到达大学城校区及国际校区。

- **耗时**：五山至大学城音乐厅约 45~60 分钟，至国际校区也在此区间内，早晚高峰可能延长
- **班次**：工作日班次较休息日更多

## 休闲娱乐

- **天河新天地**：位于天河客运站附近，有商场、电影院等设施。
- 五山地铁站周边的娱乐去处欢迎有了解的同学补充。
