# Copyrights

This page summarizes the licensing status of the project text, attachments, third-party assets, and embedded resources.

All original content in this repository is now uniformly published under **CC0 1.0 Universal (CC0 1.0)**. You may copy, modify, distribute, and use the content for any purpose without requiring attribution or sharing under the same license.

When reusing material, please keep the following principles in mind:

- The root `LICENSE` is the repository-level default statement.
- If a page includes its own source, restriction, or exception note, that page-level note should take priority.
- Third-party images, PDFs, embedded tools, and external resources should not be assumed to share the same license as the handbook text.
- If reuse, redistribution, or commercial use matters, verify the original source instead of relying on a single sentence in this repository.

## Third-Party Assets

Third-party assets referenced in this project (including but not limited to images, PDFs, embedded components, external links, and interactive tools) retain their copyright with their respective owners.

Guidelines:

- **Attribution-only**: This project references third-party assets solely for reader convenience. It does not claim ownership of their content, nor do they fall under the CC0 1.0 license of this repository.
- **Unspecified license**: For third-party assets without an explicit license, **CC BY-SA 4.0** is assumed by default. If the copyright holder considers this assumption incorrect, please open a GitHub Issue and the maintainers will address it promptly.
- **Verify before reuse**: When reusing third-party assets found in this repository, verify their license terms at the original source. This repository makes no legal guarantee regarding their licensing status.
- **Copyright concerns**: If you believe any content in this repository infringes your copyright, please submit a GitHub Issue with details. The maintainers will review and remove or replace the content as soon as possible.

## Contributor License

**Original** content submitted to this repository (including but not limited to text, images, tables, and scripts) is published by default under the same **CC0 1.0 Universal (CC0 1.0)** license as the repository itself.

Key terms:

- **No opt-out**: This project explicitly does not accept original content that is not licensed under CC0 1.0. By having their original content merged via a Pull Request, contributors agree to license it under CC0 1.0.
- **Third-party content**: If a submission includes non-original third-party material, the contributor must ensure its license is compatible with this project and must note the source and license in the submission.
- **Maintainer review**: Before merging, maintainers should check whether the submission contains third-party material without proper attribution. If potentially infringing content is found, maintainers should ask the contributor to clarify or remove it before merging.
