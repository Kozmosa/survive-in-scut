# E区宿舍概况

## 1. 硬件

### a. 宿舍内部

宿舍内部虽然不大，但是由于是新建的，设施都比较新。

![宿舍全景概览](/assets/hemc/dorm_e_overview.jpg)

**基本规格：**

* 宿舍普遍是 **5m × 3.6m** 的四人间
* 上床下桌，有自己的衣柜、书架和放行李的架子

**书桌：**

![书桌与储物柜](/assets/hemc/dorm_e_desk.jpg)

* 规格：**1.2m × 0.6m**
* 一侧与上床的楼梯相连，另一侧和衣柜相连
* 上方后部有书架，需要合理规划空间
* 书架底部与书桌高度差为 **0.6m**

**衣柜：**

* 尺寸约为 **0.8m × 0.6m × 1.15m**
* 下方还有一个宽长一样的储物柜：**0.8m × 0.6m × 0.45m**
* 衣柜旁有四个开放柜（3个在上面的尺寸 **0.3m × 0.25m × 0.33m** + 一个最下面的 **0.3m × 0.25m × 0.65m**）

**右侧储物：**

* 可以放置生活用品和私人物品的抽屉和柜子
* 抽屉：**0.4m × 0.33m × 0.1m**
* 柜子：**0.6m × 0.33m × 0.45m**

**书柜：**

* 头上三个书柜，每个 **0.4m × 0.2m**（不封顶）

**楼梯柜：**

* 左/右侧是上床的楼梯，楼梯上有两个可储存柜子
  * **0.5m × 0.65m × 0.25m**
  * **0.5m × 0.4m × 0.25m**
* 因为两个人共用一个楼梯，建议和室友商量使用

**床铺：**

* 尺寸：**1.9m × 0.9m**
* 学校会自动加装护栏和挂蚊帐的架子

### b. 阳台及厕所浴室

* 阳台通常是 **1.8m × 2.15m**（某些宿舍的一角会被墙体占用）
* 有洗漱台、洗漱架
* 卫生间和浴室**干湿分离**，分别约为半个阳台大小
* 水费：**3.75元/吨**
* 电费：**0.6295元/度**

## 2. 软件

* 宿舍配备**空调**、**风扇**
* **饮水机**在每层楼固定位置
* **不熄灯不断电**（会断校园网）
* 每栋楼应该是 **7部电梯**
* **e2/e3** 有地下车库以及直达的楼梯（一部到车库的电梯）
* 楼下有**自助售货机**和**外卖点**（校内食堂外卖或者西亚超市的可以配送到这里）
* 每栋楼下有自己独特的**功能室和活动室**，例如：
  * **e2**：厨房、书法室
  * **e3**：心理辅导室

## 3. 交通

E区确实离各个校内地方比较远，以下描述去学校一些重要地方的路径和时间（以 **e3** 为例，e2会近50m左右，e1会近100m左右）：

| 目的地                             | 距离/时间       | 备注                                     |
| ---------------------------------- | --------------- | ---------------------------------------- |
| **教学楼 a5~a1**                   | 500m ~ 700m     | 由远到近；a1高楼尽量骑车，提前至少15分钟 |
| **图书馆**                         | 约800m          | 紧挨着a1                                 |
| **教学区操场/体育馆**              | 约800~900+m     | 通勤约20分钟                             |
| **实验楼（b开头）**                | 15~25分钟       | 各栋楼不等                               |
| **快递点**                         | e3楼下 / 二饭旁 | 偶尔会有c10楼下的                        |
| **外卖（北门）**                   | 约8分钟         | 拿取较为方便                             |
| **穗石**                           | 约15分钟        | 附近美食和商圈                           |
| **三饭**                           | 约15分钟        |                                          |
| **二饭旁（瑞幸/遇见小面/麦当劳）** | 约10分钟        |                                          |
| **gogo新天地**                     | 约25分钟        |                                          |
| **南亭**                           | 约30分钟        |                                          |
| **北亭村**                         | 约40分钟        |                                          |
| **生物岛**                         | 50分钟以上      | 建议不骑车                               |
| **地铁站（大学城站）**             | 约20分钟        | 从北门出发到d口；沿学校内部从西门出到f口 |

> **Tip**：从f口回e区骑车一路下坡比较爽！
