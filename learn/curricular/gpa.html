GPA ​

一句话区分 ​

概念本质主要用途关键词
均分 / 百分制平均学分绩点课程百分制成绩按学分加权后的结果校内推免、转专业等首考、必修、百分制
GPA / 四分制平均学分绩点课程绩点按学分加权后的结果，绩点由成绩分布或等级换算得到出国成绩证明、学士学位资格等最高成绩、4.0、动态换算
综测分 / 综合测评总积分学业成绩、操行评定、综合素养的年度综合评价奖学金、三好学生、优秀学生干部等评奖评优班级排名、加分、公示

均分：校内竞争最常用的硬指标 ​

正式名称是百分制平均学分绩点。学生口中的“均分”大多数时候指的就是它，而不是简单把几门课分数相加除以门数。

计算方式：

text
百分制平均学分绩点 = Σ（课程百分制成绩 × 课程学分数） / Σ课程学分数

核心规则：

  • 主要用于校内推荐免试研究生、转专业等场景。
  • 计算范围是主修专业综合培养计划要求的必修课程。
  • 按学生首次考核取得的课程成绩计算。
  • 不及格、不通过、重修通过、重考通过的课程，百分制成绩按 0 计。
  • 五级制折算：优秀 95，良好 85，中等 75，及格 65，不及格 0。
  • 两级制折算：通过 80，不通过 0。
  • 经批准的缓修或缓考通过课程，可以计入修读学期的百分制平均学分绩点。

所以，校内竞争里的“高均分”不是修水课刷出来的平均数，而是必修课首考成绩按学分加权后的结果。高学分课就是杠杆，挂科和重考就是伤口，后面补得再漂亮，也很难在这套账里复活。

GPA：对外证明和学位资格的 4.0 制账本 ​

本科生手册里的正式说法是四分制平均学分绩点（GPA）。

计算方式：

text
GPA = Σ（课程绩点 × 课程学分数） / Σ课程学分数

核心规则：

  • 主要用于申请出国开具成绩证明等场景。
  • 计算范围是主修专业综合培养计划要求的课程；通识教育课程、辅修课、校外课程等不纳入计算。
  • 按学生考核取得的最高课程成绩计算。
  • 重考成绩绩点认定区间以原班级正考成绩区间为准，但封顶 3.3。
  • 学士学位授予要求之一：主修专业综合培养计划内全部课程（辅修课程、校外课程除外）的四分制平均学分绩点须达到 2.0 及以上。

百分制课程的 GPA 不是固定的“90 分 = 4.0”式换算，而是根据课程班成绩分布动态转换：

百分制课程成绩位置等级课程绩点
前 20%A4.0
20.1% - 35%B+3.7
35.1% - 50%B3.3
50.1% - 60%B-3.0
60.1% - 70%C+2.7
70.1% - 80%C2.3
80.1% - 90%C-2.0
后 10%D1.7
不通过F0.0

等级制课程换算如下：

计分方式课程成绩等级课程绩点
五级制优秀A4.0
五级制良好B3.7
五级制中等C2.7
五级制及格D1.7
五级制不及格F0.0
二级制通过P3.0
二级制不通过F0.0

这套规则的要害是：GPA 更像“对外翻译系统”，不是校内推免排序的主战场。你要出国，它重要；你要保研或转专业，先看百分制平均学分绩点。

综测分：评奖评优的混合账本 ​

综合测评总积分不是纯成绩。它是一个把学习成绩、操行、竞赛、论文、文体活动、劳动教育等内容揉在一起的年度评价系统。

总公式：

text
H = 0.7X + 0.1C + 0.2S

其中：

  • H：综合测评总积分。
  • X：学业成绩积分，按学年必修课（含体育课）和院系可自行选定添加的选修课成绩百分制加权平均分计算。
  • C：操行评定积分，公式为 C = C1 + C2，包括操行评定基本分和操行评定加分。
  • S：综合素养评定积分，公式为 S = D + 3Z + T + M + L，包括德育、智育、体育、美育、劳育。

学业成绩积分 X 的计算规则和“均分”高度相似：按学分加权，五级制和两级制按当量折算，首次考试成绩计算，不及格、不通过以及重考、重修通过的课程绩点数为 0，经批准缓修通过的课程可计入修读学期。

综测分的主要用途是评奖评优。例如：

  • 国家奖学金：综合测评成绩总积分和学业成绩积分原则上都要居班级前 10%。
  • 国家励志奖学金：综合测评成绩总积分居班级前 33%，学业成绩积分居班级前 38%。
  • 学校一等奖学金：综合测评成绩总积分居班级前 10%，学业成绩积分居班级前 15%。
  • 学校二等奖学金：综合测评成绩总积分居班级前 20%，学业成绩积分居班级前 25%。
  • 学校三等奖学金：综合测评成绩总积分居班级前 33%，学业成绩积分居班级前 38%。

综测不是“成绩不好也能靠活动翻盘”的许愿池。它的权重结构很清楚：X 占 70%，成绩仍然是主体；活动、竞赛、荣誉、社会工作是放大器，不是发动机。

该盯哪个指标 ​

目标优先盯的指标判断逻辑
转专业百分制平均学分绩点、关键课程单科成绩很多学院直接看必修课均分、数学/英语/物理等单科门槛
保研百分制平均学分绩点本科生手册明确推免基本条件使用前三学年必修课原始成绩百分制平均学分绩点排名
出国申请四分制 GPA、成绩单呈现GPA 是成绩证明的对外语言，但不同学校还会自行重算
学士学位四分制 GPA主修专业综合培养计划内课程 GPA 需达到 2.0 及以上
奖学金/三好/优干综合测评总积分 + 学业成绩积分既看综测总排名，也看学业成绩积分排名，且通常要求无挂科、体测达标

校内硬通货是必修课首考加权均分；出国硬通货是 GPA；评奖硬通货是综测排名。三套账分别优化，别拿一套账去赌另一套账。
